## Professional AI Resume Builder

ResumeAI helps you build ATS-compliant resumes, enhance them with Google Gemini AI, and increase your chances of getting shortlisted.

## Features
- **Resume Builder**: Step-by-step builder with real-time preview.
- **AI Enhancer**: Polish descriptions and summaries using Gemini AI.
- **ATS Checker**: Verify your resume against job descriptions.
- **PWA Support**: Install as a native app on mobile and desktop.
- **Firebase Integration**: Secure authentication and cloud persistence.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`

2. Set up Environment Variables:
   Rename `.env.local.example` to `.env.local` and add your Firebase and Gemini API keys.

3. Run the app:
   `npm run dev`
